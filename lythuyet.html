Câu 1: position absolute và relative

relative: định vị theo vị trí ban đầu, vẫn chiếm chỗ

absolute: định vị theo phần tử cha, không chiếm chỗ

Ví dụ:

.container { position: relative; } .box { position: absolute; top:0; right:0; }

Ứng dụng: đặt icon, popup, layout

Responsive: nếu dùng px dễ lệch → nên dùng %, flex, grid

Câu 2: Domain - Hosting - DNS

Quy trình:

  • Mua domain
  • Thuê hosting

DNS: chuyển domain → IP server

Công cụ:
